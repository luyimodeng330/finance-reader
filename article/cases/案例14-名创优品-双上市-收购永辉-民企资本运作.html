案例背景｜案例来龙去脉：一个极致效率零售商，如何用 IP 与资本两次跨越

先把这个故事的骨血讲透。没有它，后面的"视角"都是空中楼阁——而这个故事最有价值的，是一个被叫作"十元店"的品牌，怎么先后拿下美股、港股，又转头"抄底"下一个零售巨头。

1.1 来路：叶国富的"低价美学"（2013–2019）

2013 年，叶国富在广州创立名创优品，模式是"优质低价 + 设计感 + 加盟"：用无印良品式的审美，卖十分之一的价格。他早年在"哎呀呀"饰品店积累的，是对"高频低价 + 加盟扩张"的直觉。2015 年启动全球化，正式出海；到 2019 年，全球门店超 4200 家、覆盖 80 多个国家和地区，连哈佛旁边都开了店。风口之上的隐忧也有：早期"山寨""侵权"的争议，和加盟商管理的复杂度。

1.2 蜕变：双上市，把"十元店"做成上市公司（2020–2022）

2020 年 10 月 15 日，名创优品在纽交所上市（代码 MNSO），发行价 20 美元，IPO 募资约 6.2 亿美元，估值约 70 亿美元——成立仅 7 年。关键动作在 2022 年 7 月 13 日：它在港交所完成双重主要上市（9896.HK），注意是"双重主要"而非"二次上市"——意味着在两地都拥有独立上市地位。资本逻辑很清楚：中概股在美股估值受压，回港双重主要上市等于多开一扇融资的门、贴近本土投资者。

1.3 节奏：从卖货到卖 IP（2020–2024）

上市后战略重心转移：从"便宜好物"升级为"IP 设计零售"，密集联名 Disney、三丽鸥、Chiikawa 等，IP 产品成了增长引擎。门店也从"十元店"进化成"生活好物集合店"，客单价被拉高。到 2024 年，全球门店超 7000 家（据公开报道），并孵化出潮玩子品牌 TOP TOY。它卖的不再只是商品，是"情绪价值 + 高频进店"。

1.4 成长弧：营收与利润的双线

财务上，2020 财年营收约 89 亿元人民币，到 2024 财年约 170 亿元人民币（据公开报道，约翻倍）；净利润转正并持续增长，海外营收占比逐年提升。这条曲线说明它不只是在"开更多店"，是在把单店效率和 IP 溢价同时做厚。

1.5 资本弧：63 亿"抄底"永辉（2024.9）

2024 年 9 月 23 日，名创优品公告：旗下骏才国际以约 62.7 亿元人民币（约 8.99 亿美元）收购永辉超市 29.4% 股权（受让自牛奶公司、京东等老股东），成为第一大股东；交易单价约每股 2.35 元，而永辉当时股价约 2.3 元、已破净。叶国富的逻辑被媒体概括为"豪赌胖东来模式"——永辉正学习胖东来调改，名创想用供应链 + IP 去赋能线下商超。后续：2024 年 12 月经营者集中获无条件批准；据证券时报报道，名创账面浮盈一度约 87 亿元（永辉股价后续上涨）。但市场当天用脚投票：公告次日名创优品美股跌超 18%，投资者担心它"不务正业"。

1.6 竞争弧：它赢在"效率 × IP"

横向看，无印良品太贵、NOME 太同质、泡泡玛特只做潮玩、永辉这样的传统商超效率偏低。名创优品赢在"极致供应链效率 × IP 情绪价值"的组合拳——用低价高频把人引进门，用 IP 把人留下来、把客单价提上去。这也是它敢去"改造"永辉的底气：它认为自己手握线下零售的新方法论。

1.7 悬念：入主永辉之后

最新进展里，永辉的"胖东来式"调改门店客流确有改善（据公开报道），但整体盈利修复漫长，市场对其"十元店能否改造巨无霸商超"仍存疑。名创自己则继续扩张海外、押注 IP。一个"效率零售"能否真正重塑"低效零售"，悬念还开着。

▼ 分支 A · 向内做企业（视角向内）

0. 你为什么读这个：把别人"双上市 + 跨界并购"的资本节奏，变成自己的路线图

一个做"十元店"的，能先后拿下美股、港股、再吃下永辉，靠的不是运气，是"节奏感"。

我读它的一个感受：名创最值得学的，是它每一步资本动作都服务一条主线——把"效率"做厚、把"IP"做重。很多人融资是为了活下去，它融资是为了"换赛道"。你对照的，应该是自己的资本节奏，不是它的门店数。

2. 名创这一步步，对你做企业意味着什么

① 双上市不是终点，是"融资渠道冗余"

美股 + 港股双重主要上市，等于在两个池子都有身份。对你：你融资不要只盯一个市场，留多条路，环境一变你还有后手。单地依赖，是很多企业在资本寒冬里最被动的地方。

② "不务正业"的并购，先看协同

买永辉表面跨界，实质是用 IP + 供应链赋能线下。对你：你跨界或并购，协同点写得出吗？写不出就是分散注意力，写得出才是第二曲线。

③ IP 是资产，不是噱头

联名 IP 被它做成了增长引擎，而不是一次性营销。对你：你有没有把"流量合作"沉淀成"复购资产"？很多品牌蹭完热度就散了，它把热度变成了货架。

④ 股价跌是市场给你定价，不是给你定罪

公告后跌 18%，但账面浮盈。对你：别被短期市场情绪绑架长期判断——只要你认准的协同逻辑成立，一时的估值波动不是决策依据。

3. 你自己的企业，可以对照这四步

同一个案例，落到"怎么做企业"上，是四个可执行的动作，不是四个抽象层级：

  1. 切口：你上市或出海的第一刀，是"补估值"还是"补渠道"？名创双上市补的是"融资冗余"，方向很明确。
  2. 资产：你最值钱的"不可替代资产"是 IP 心智、供应链，还是门店数？分清了，才知道该并购谁、不并购谁。
  3. 控制权：你拿永辉 29.4% 当"第一大股东但不控股"，用影响力而非全资——控制权可以是"关键少数"，不一定要 51%。
  4. 跃迁：名创从"卖货"跳到"卖 IP + 生态"，这是跃迁。你的企业，有没有从"产品"跳到"平台 / 生态"的路径？

4. 合上文章，想想你自己的四件事

这篇没有标准答案，留四个问题给你自己：

  1. 你的融资，是不是只依赖单一市场或单一投资人？万一环境变了，你的后手在哪？
  2. 你若想跨界或并购，协同点写得出来吗？还是只是"看着热闹"？
  3. 你手里的"流量合作"，有没有沉淀成可复购的资产？还是蹭完就散？
  4. 你做资本决策，是看长期协同，还是被短期股价牵着走？

5. 相关映射：用框架看一个 IP 资本化样本（思维训练）

和名创形成有趣对照的，是 泡泡玛特：同样把"IP"做成核心资产、同样走向资本化，但路径相反——一个从"效率零售"长出自有 + 联名 IP，一个从"潮玩 IP"长出自有渠道。用第 3 节的"切口 / 资产 / 控制权 / 跃迁"四步，去拆解泡泡玛特（或你自己的品牌）：

  1. 它的"切口"是 IP 还是渠道？和你可能的切口，打法差在哪？
  2. 它的"不可替代资产"是 IP 心智还是供应链？和名创的效率壁垒，相同不同在哪？
  3. 它的"控制权 / 资本化节奏"是快是慢？你更想学哪条路？
  4. 如果用这四步写一段"我的品牌诊断"，你会怎么写？最该补的是哪一步？

这套四步不只对零售品牌有效。任何"产品 + 情绪"的生意，都能这么拆。训练几次，你看自己企业的资本眼光会不一样。

6. 我读完后自己的一点想法

写完这篇，我最触动的不是 63 亿的并购，而是那句藏在背后的逻辑：一个人 / 一家企业最值钱的，从来不是被看见，是被需要且不可替。名创最值钱的，是"让年轻人想进店、让 IP 方想合作"——它被渠道需要、被消费者需要、被合作方需要。这中间的认知，才是它敢去改造永辉的底线。你做自己的企业，最该守住的，也是这句：先把"被需要且不可替"的那一点磨出来，再谈放大和跨界。

▼ 分支 B · 向外连客户（视角向外）

0. 你为什么读这个：用一个好案例，跟客户聊到一起去

高净值客户不缺一个"卖产品的人"，她缺一个能跟她在同一个资本语境里聊世界的人。

客户心声 你的门户里有一句客户证言："以前看财报像天书，现在能跟家里管钱的先生平起平坐地聊资产配置了。"这句话泄露高净值女性最深的两层心理：第一，她长期被"降维"对待——先生、banker、顾问都默认她不懂；第二，她渴望的不是被服务，是被"平视"。

名创这个案例，我的一个感受是：它其实是一张体面获客的入场券——你能用它证明"我和你在同一认知语境里"，而不是靠推销硬闯。不是"我懂比你多"的俯视，也不是凑上去讨好，是"我也在这个资本世界里，咱们对等聊"。

2. 这个案例里，有哪些点能让你跟客户聊出深度

① 70 亿估值到"抄底"永辉——民企的资本野心

很多客户一听"十元店买下超市巨头"觉得跨度太大。你能接住这句话，带她看另一面：这是"效率零售"对"低效零售"的降维，不是乱花钱。你这么一拆，客户立刻知道你看得比热搜深。

② 双上市 = 融资冗余的安全感

跟有海外资产的客户聊：两个上市地等于两扇门，永辉式"单地依赖"才是雷。这个"不要把鸡蛋放一个资本市场"的角度，比任何配置话术都更"懂老板的难"。

③ 跨界并购的协同逻辑

用名创买永辉，你能聊到客户自己："您自己的跨界投资，协同点写得出吗？"带她识别"分散"和"第二曲线"的差别。

④ IP 资产的资本化

"情绪价值怎么变成估值"，这个点很适合跟关注消费、文娱赛道、或自己有品牌资产的客户聊，顺手把话题引到她的配置思路上。

3. 你跟客户聊资本，可以用的一个框架

同一个案例，你带客户从浅到深，是四层递进，不是四个层级评判。目的是让她觉得"你带我一层层看透了"：

  1. 先聊事件：一个"十元店"品牌，先后上了美股、港股，又花 63 亿买下永辉。——破冰，谁都接得上。
  2. 再聊逻辑：为什么敢买？因为它手握"效率 + IP"，想用这套方法论改造线下商超。——展示你懂行，不止追热点。
  3. 聊资本视角：它用 29.4% 当"关键少数"大股东，不控股却有影响——展示你深，能看到老板做资本的巧劲。
  4. 聊资本与权力：从"卖货"到"用资本重塑一个行业"，真正的权力是方法论的输出，不是持股比例。——让客户觉得你看得透，聊到了别人聊不到的层。

4. 下次跟客户聊，你可以试着抛的几个问题

这篇没有标准话术，留几个你能自然抛出的问题：

  1. 名创"双上市"留了两扇融资门——您自己的资产，有没有类似的"冗余安全感"？
  2. 用刚才那个"四层框架"，您给自己的资本认知打到第几层？身边很多人其实停在第一层（只看估值）。
  3. 如果一位老板问您"这笔跨界买得值不值"，您会拿什么证据平起平坐地聊，而不是附和？
  4. 名创把"IP"做成资产——您自己手里，有没有被低估的"情绪 / 品牌"资产？

5. 相关映射：用框架聊一个 IP 资本化样本（思维训练）

跟客户聊"对照组"，泡泡玛特 和名创几乎是硬币两面：一个从 IP 长渠道，一个从渠道长 IP，殊途同归都走向资本化。用第 3 节的"事件 → 逻辑 → 资本视角 → 资本与权力"框架，跟客户聊泡泡玛特：

  1. 泡泡玛特这个事件，大众现在认知在第几层？它走到"资本与权力"这层了吗？
  2. 它怎么把"IP"做成估值？和名创的"效率 + IP"对照，给客户看消费赛道的两种打法。
  3. 它和名创的"控制权 / 资本化节奏"差在哪？您更想学哪条？
  4. 您用这个框架，能给泡泡玛特写一段"资本心智分析"吗？写得出，说明框架真用熟了。

这个框架不只聊零售。任何"产品 + 情绪 + 资本"的故事，都能这么带客户一层层拆。练熟了，你和她聊的就不只是新闻，是认知。

6. 我读完后自己的一点想法

写完这篇，我最想跟客户分享的不是 63 亿的数字，而是那种"节奏感"背后的人与资本：真正站得住的资本动作，从来不是追热点，是看清自己被需要的地方、再用资本把它放大。你带客户看透这一层，她记住的就不是"你又给我讲了个案例"，而是"你是那个能跟我对等聊世界的人"。这，才是这类案例在你手里最值钱的地方。
