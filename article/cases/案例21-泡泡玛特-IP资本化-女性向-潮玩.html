泡泡玛特：从盲盒到IP帝国，Labubu 重启增长的资本故事

案例背景｜案例来龙去脉：一个靠"盲盒"起家、被 Labubu 一把拉回千亿市值的IP样本

先把这个故事的骨血讲透。没有它，后面的"视角"都是空中楼阁——而这个故事最迷人的，是它证明了：当物质过剩，人们愿意为"情绪、审美、身份"付溢价，而且付得相当慷慨。

1.1 来路：从泡泡杂货店到潮玩第一股（2010–2020）

王宁 2010 年在北京创立泡泡玛特，最早是潮流杂货店；2015 年转型聚焦潮玩，签下 Molly，2016 年推盲盒，引爆"收集欲"。2020 年底港股上市（09992.HK），发行价 38.5 港元，彼时市场把它当"新消费明星"。

1.2 诞生分歧：它卖的不是玩具，是"情绪"

泡泡玛特把"无用的小玩具"卖成"有身份感的收藏"，核心客群是女性与年轻世代。它证明：当功能过剩，情绪、审美、身份才是稀缺品。这是它整个估值逻辑的底座——卖的是感觉，不是塑料。

1.3 上线即爆：盲盒经济学

盲盒用"未知 + 收集 + 社交"制造复购，Molly、Dimoo 等 IP 奠定基本盘。2020 上市时，市场为这套"成瘾机制"给足了想象空间，市值一度冲高。

1.4 成长弧：2024 被 Labubu 点燃

2024 年营收 130.4 亿元（+106.9%），经调整净利润 34.0 亿元（+185.9%）；2025 年营收约 371.2 亿元（+184.7%）。增长引擎是 Labubu（艺术家龙家升创作的 IP）全球爆火——从东南亚火到欧美，二手溢价数倍，拍卖会百万成交。一个 IP，把公司从增长焦虑里一把拉了回来。

1.5 资本弧：市值过山车

据公开报道，2025 年中市值一度突破 3500 亿港元（6 月 12 日约 3583 亿港元），较发行价累计涨数倍；随后随业绩与预期波动，2026 年初回落至约 2500 亿港元区间（2026.3.25 公布业绩后单日蒸发超 200 亿、约 2485 亿港元）。市值在千亿至三千亿+港元间剧烈摆动，被 Labubu 重启后屡创新高又回落。

1.6 竞争弧：它赢在"IP 运营"，不止"盲盒"

当市场以为泡泡玛特只是盲盒，它用 Labubu 证明自己是 IP 运营公司：签艺术家、做内容、做全球渠道。对比单纯卖玩具的，它的护城河是"持续造星 IP"的能力——这才是资本市场真正买单的部分。

1.7 悬念：Labubu 之后，下一个是谁

单一 IP 依赖是风险，Labubu 终有热度周期。泡泡玛特能否再造下一个现象级 IP、把"盲盒公司"彻底变成"IP 帝国"，决定它市值能不能稳在千亿之上。这又回到了 IP 公司的终极考题：你是在运营 IP，还是在赌一个 IP。

点上面的目标，查看对应分支的完整后半部分（案例背景已读完，下面这段是只属于你这个目标的纵深）。

▼ 分支 A · 向内做企业（视角向内）

0. 你为什么读这个：把别人的弯路，变成自己的捷径

泡泡玛特最值得琢磨的，是它怎么把"没用的小东西"卖成"被争抢的资产"。对做任何面向女性、面向年轻客群生意的人，这是一本活教材——它讲的是"情绪怎么定价"。

2. 案例主角这一步步，对你做企业意味着什么

① 卖情绪比卖功能值钱

女性愿意为审美和身份付溢价，功能反而最不稀缺。你的产品，有没有"非功能"的那层价值？

② 盲盒是机制，不是产品

复购靠"未知 + 收集"，不是靠东西本身好用。你能不能给你的生意设计一套"让人想再来"的机制？

③ IP 要"造星"而非"买断"

签艺术家、做内容，才能持续产出。买一个现成 IP，热闹一时；建一套造星机制，才长久。

④ 警惕单一 IP 依赖，持续增长靠 IP 矩阵

Labubu 救了它，也可能困住它。你的增长，是绑在一个爆款上，还是长在一组 IP 上？

3. 你自己的企业，可以对照这四步

  1. 切口：你有没有一个"非功能性但被强烈需要"的情绪入口？泡泡的切口是"收集欲 + 审美认同"。
  2. 资产：你的资产是 IP，还是产能？泡泡的资产是 IP 与艺术家关系，这才是它值钱的地方。
  3. 控制权：你怎么锁定艺术家与 IP？独家签约加内容运营，把核心创意攥在自己手里。
  4. 跃迁：单一 IP 之后，怎么矩阵化、全球化？Labubu 出海，就是一次跃迁示范。

4. 合上文章，想想你自己的四件事

  1. 我的产品，卖的是功能，还是情绪？
  2. 我的复购，靠产品力，还是靠机制？
  3. 我的核心资产，是我能控制，还是易被挖走？
  4. 我的增长，靠一个爆款，还是一组 IP？

5. 相关映射：用框架看一个国内品牌（思维训练）

对标：名创优品（IP 联名 + 出海）。它同样做"情绪 + 性价比 + 全球化"，但路径是"联名成熟 IP"而非"自造 IP"，恰好能照出"造星"与"借星"两种打法的取舍。

  1. 切口：泡泡切"收藏与身份"，名创切"平价好物 + 联名惊喜"。
  2. 资产：泡泡重自有 IP，名创重渠道与联名网络。
  3. 控制权：泡泡锁艺术家，名创锁供应链与门店。
  4. 作业：用"切口 / 资产 / 控制权 / 跃迁"四步，拆解为什么泡泡估值能压过传统玩具商。

据公开报道，名创优品以 IP 联名与全球化见长，可作为本案例的对照样本。

6. 我读完后自己的一点想法

落到"守住被需要且不可替"：泡泡被需要，是因为它把"无用之美"做成了身份表达。它的不可替，是持续造 IP 的能力。任何面向人的生意，先把"被情感需要"做厚，资本才会追——别只学它卖盲盒，要学它造情绪。

▼ 分支 B · 向外连客户（视角向外）

0. 你为什么读这个：用一个好案例，跟客户聊到一起去

女性客户、年轻客户里，几乎没人没听过 Labubu。泡泡玛特是一个天然能聊出共鸣的资本案例——它讲的是"情绪怎么变成钱、IP 怎么变成市值"，而这个话题，客户自己就带着感受。

2. 这个案例里，有哪些点能让你跟客户聊出深度

① 情绪消费是真实的大生意，不是小打小闹

一个塑料娃娃撑起三千亿市值，足以刷新客户对"非刚需"的认知。

② 盲盒机制制造复购，是商业设计

帮客户看懂：让人"上瘾"的，不是东西，是机制。这层能聊出消费心理的纵深。

③ Labubu 全球爆火，说明中国 IP 能出海

从东南亚到欧美，二手溢价数倍——这是关于"文化输出"的好话题，客户容易有共鸣。

④ 市值过山车，提醒"单一 IP 依赖"的风险

涨到 3500 亿又回落，帮客户理解：再火的 IP，也有热度周期，估值要打折看。

3. 你跟客户聊资本，可以用的一个框架

  1. 先聊事件：泡泡玛特 2024 靠 Labubu 营收翻倍、市值一度破 3500 亿港元，后又回落至约 2500 亿。
  2. 再聊逻辑：卖情绪 + 盲盒机制 + IP 运营，让它从玩具商变 IP 公司。
  3. 聊资本视角：市场为"增长确定性"给高溢价，但单一 IP 热度周期带来估值波动。
  4. 聊资本与权力：谁掌握 IP 与艺术家，谁就掌握定价权——泡泡用独家签约锁定，这层很适合聊"无形资产的控制权"。

4. 下次跟客户聊，你可以试着抛的几个问题

  1. 您身边的"情绪消费"，有没有被您当成生意看？
  2. 您觉得一个玩具凭什么卖到二手溢价数倍？
  3. 您的品牌，有没有能"出海"的 IP 基因？
  4. 如果您的核心靠一个爆款，风险怎么管？

5. 相关映射：用框架聊一个国内品牌（思维训练）

对标：名创优品（IP 联名出海）。同样做"情绪 + 全球化"，一个自造 IP、一个联名 IP，正好用四层框架帮客户看清"情绪消费怎么资本化"。

  1. 事件：泡泡自造 Labubu 引爆，名创靠联名扩圈。
  2. 逻辑：情绪溢价 + 机制复购，是两类公司共同的底层。
  3. 资本视角：市场为"可复制的增长"给溢价，为"单一依赖"打折。
  4. 资本与权力：IP 的控制权在谁手里，谁就掌握定价——用框架带客户看"无形资产怎么变成市值"。

据公开报道，名创优品以 IP 联名与全球化见长，可作为本案例的对照样本。

6. 我读完后自己的一点想法

落到"你是她对等聊世界的人"：聊泡泡玛特，不是显摆你知道 Labubu，而是帮客户看懂"情绪、IP、市值"这条价值链。你能把潮流讲成资本逻辑，而不是只聊娃娃多可爱，你就是她对等聊世界的人。
