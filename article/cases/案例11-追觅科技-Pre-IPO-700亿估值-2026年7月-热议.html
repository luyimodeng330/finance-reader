追觅科技 Pre-IPO：700 亿投前估值、3.5 亿单笔门槛，冲刺港股

案例背景｜案例来龙去脉：一个扫地机器人龙头，怎么把 Pre-IPO 做成一场"高门槛抢筹"

先把这个故事的骨血讲透。它最值钱的，不是"700 亿"这个数字，而是追觅怎么用"硬技术 + 出海 + 高门槛"，把一轮融资变成了机构排队抢的稀缺品。

1.1 来路：从清华宿舍到高速马达死磕（2015–）

追觅科技由俞浩于 2015 年创立，团队脱胎于清华"天空工场"，技术底色是自研高速数字马达——转速做到每分钟十万转以上量级，这是扫地机器人、无线吸尘器、高速吹风机的"心脏"。早期不跟人贴牌、不拼低价，而是死磕这颗马达，把"别人做不了的核心部件"攥在自己手里。这一步，决定了它后来能走出一条和多数国货不同的路。

1.2 诞生：死磕核心技术，错位切入（2017–2020）

从扫地机器人切入，追觅没有走"互联网营销 + 代工"的捷径，而是用自研马达做差异化性能。它先打海外市场（尤其欧洲），避开国内红海价格战，用"中国智造 + 高性能"建立心智。这一阶段的关键词是"技术即壁垒"——当别人在拼谁更便宜，它在拼谁更强。

1.3 引爆：智能清洁赛道爆发 + 多品类出海（2020–2024）

居家经济与"懒人经济"叠加，扫地机、洗地机、高速吹风机多线齐发，海外占比持续走高。追觅成为"中国硬科技出海"的代表之一，和石头、科沃斯、云鲸同台，却靠"技术 + 出海 + 多品类"走出自己的弧线。据公开报道，其估值在这几年从早期数亿美元一路抬升。

1.4 成长弧：从独角兽到 Pre-IPO 标的（2021–2025）

多轮融资后，追觅稳居智能清洁第一梯队。据公开报道，其估值在 2024 前后已达数十亿美元量级，跻身超级独角兽。增长曲线、出海收入、技术储备，让它从"好公司"变成"资本想进的标的"——这一步，是所有硬科技企业的梦想终点。

1.5 资本弧：2026 开 Pre-IPO 窗口，投前 700 亿、门槛 3.5 亿（2026.6）

2026 年 6 月初，追觅开放直接融资窗口，据多家媒体报道：投前估值约 700 亿元人民币，单笔投资最低门槛 3.5 亿元，计划释放 5%–10% 的 Pre-IPO 份额，融资窗口预计 7 月初关闭。申报机构已达数十家，含中东主权基金与大型上市公司；据公开报道，早期股东小米已约 200 倍回报套现离场。市场传其目标市值约 1500 亿元，并于 6 月 12 日前后传出赴港上市筹备消息。高估值 + 高门槛，把一轮融资做成了"抢筹"。

1.6 竞争弧：它赢在"技术 + 出海"，也困在"价格战"

和石头（技术派）、科沃斯（渠道品牌派）、云鲸（单品爆款派）相比，追觅的差异化是自研马达带来的性能壁垒 + 出海占比高。但智能清洁赛道的宿命是同质化与价格战——当性能差距收窄，"护城河"还能守多久，是所有玩家的共同疑问。追觅用多品类扩张对冲，但每跨一条线，就多一个对手。

1.7 悬念：700 亿估值的三道坎（2026 之后）

融资窗口开启前后，市场也传出争议：有报道指追觅在开放融资阶段裁撤部分 BU、以及"崩老头"等舆论风波。悬念留在这里——高估值能否被二级市场接住？赴港上市时间表是否如期？增长是否可持续？它是一个"硬科技标杆"，还是一轮"Pre-IPO 估值狂欢"，要等敲钟那天由市场定价。

▼ 分支 A · 向内做企业（视角向内）

0. 你为什么读这个：把别人的融资节奏，变成自己的资本日历

读追觅，对你做企业的人，价值不是"羡慕 700 亿"，是"学它怎么把融资做成抢筹"。

追觅最让我触动的一点：它不是去求机构投，而是让机构排队抢——靠的是硬技术筑底、出海拉开差距、再用高门槛制造稀缺。你做自己的企业，也会走到"要不要融资、怎么融资"的节点，真正要学的，是怎么把"被需要"变成"议价权"。

2. 追觅这一步步，对你做企业意味着什么

把追觅当成主角，看它每一步动作背后的取舍。对你最有用的，不是"它多会融"，而是"它怎么把技术变成议价权"。

① 死磕核心技术，是最高级的切口

自研马达，别人三个月抄不走。对你：你有没有一个"别人做不了"的技术或痛点？贴牌能活，但握不住议价权。

② 出海错位，避开红海

国内卷低价，它先打欧洲。对你：你的增量市场在国内，还是海外？换条赛道，可能就不必和所有人挤一座桥。

③ 多品类扩张，用第二曲线对冲

扫地机 → 洗地机 → 吹风机，一条技术底座撑多品类。对你：你的第二曲线，是顺着核心能力长的，还是另起炉灶？

④ 高门槛抢筹，把融资做成稀缺

3.5 亿门槛、700 亿估值，反而让机构抢。对你：你融资时是"求人"还是"让人抢"？门槛怎么设，决定了你 Stad 什么价。

3. 你自己的企业，可以对照这四步

同一个案例，落到"怎么做企业"上，是四个可执行的动作，不是四个抽象层级：

  1. 切口：你是不是有一个"别人做不了、抄不走"的技术或痛点？像它死磕马达那样，先把切口磨成壁垒。
  2. 资产：你的"不可替代资产"是自研技术、供应链，还是品牌？追觅是马达技术——你那一点，经得起尽调吗？
  3. 控制权：你融资的节奏，是"高位拿钱、少稀释"，还是"便宜时急着拿、稀释最多"？追觅的高门槛，本质是强议价、保股份。
  4. 跃迁：从"技术公司"到"资本标的"，你离上市还差什么？底稿、治理、增长可持续——这三道坎不补，估值只是纸面。

4. 合上文章，想想你自己的四件事

这篇没有标准答案，留四个问题给你自己：

  1. 你的核心技术，别人三个月能抄走吗？抄不走的那点，你护住了吗？
  2. 你的增量市场，是国内还是海外？有没有一条更空的赛道？
  3. 你融资是求人还是让人抢？门槛怎么设，才保得住股份？
  4. 你离资本还差哪一步——底稿、治理、还是增长可持续？

5. 相关映射：用框架看一个已上市样本（思维训练）

国内有个和追觅气质互补的对照——石头科技：同样技术派、同样强出海、同样做扫地机，却更早登陆科创板、用"自研 + 高毛利"走通了"硬科技 → 上市公司"的路。一个已上市、一个冲刺中，正好看"技术怎么变成资本"。

用第 3 节的"切口 / 资产 / 控制权 / 跃迁"四步，去拆解石头（或你自己的企业）：

  1. 它的"切口"是自研导航算法，和你可能的切口，打法差在哪？
  2. 它的"不可替代资产"是技术还是渠道？和追觅的马达壁垒，相同不同在哪？
  3. 它的控制权 / 上市节奏，是快是慢？你更想学哪条路？
  4. 如果用这四步写一段"我的企业诊断"，你会怎么写？最该补的是哪一步？

这套四步不只对硬科技有效。任何"准备走向资本"的企业，都能这么拆一遍自己的议价权。训练几次，你看自己企业的眼光会不一样。

6. 我读完后自己的一点想法

写完这篇，我最触动的不是 700 亿，而是它反过来印证了那句老话：一个人 / 一家企业最值钱的，从来不是被看见，是被需要且不可替。追觅最不可替的，不是某个爆款，是那颗别人抄不走的高速马达——市场需要它，是因为它真的不可替代。你做自己的企业，最该守住的，也是这句：先把"被需要且不可替"的那点技术磨出来，再谈估值和敲钟。

▼ 分支 B · 向外连客户（视角向外）

0. 你为什么读这个：用一个好案例，跟客户聊到一起去

高净值客户不缺一个"卖产品的人"，她缺一个能跟她在同一个资本语境里聊 Pre-IPO 的人。

你的门户有一句客户证言："以前看财报像天书，现在能跟家里管钱的先生平起平坐地聊资产配置了。"这句话泄露高净值女性最深的两层心理：第一，她长期被"降维"对待——先生、banker、顾问都默认她不懂；第二，她渴望的不是被服务，是被"平视"。追觅这个案例，我的一个感受是：它其实是一张体面聊"硬科技 Pre-IPO"的入场券——很多高净值客户自己就在看一级市场、也在纠结"份额值不值"，你能用它证明"我和你在同一个认知语境里"，而不是靠推销硬闯。不是"我懂比你多"的俯视，也不是凑上去讨好，是"我也在这个资本世界里，咱们对等聊"。

2. 这个案例里，有哪些点能让你跟客户聊出深度

把追觅当成素材，看哪些点能让你跟客户从破冰聊到深。不是背数据，是借它的故事，露出你的认知。

① 700 亿估值、3.5 亿门槛——Pre-IPO 抢筹逻辑

客户若碰过一级市场，能立刻共鸣：门槛越高越抢，是稀缺在定价。你能带她看：不是"贵"，是"进不进得去"。

② 投前 700 亿 vs 目标 1500 亿——一二级价差

Pre-IPO 与上市市值的差额，是早期股东的利润池。适合聊：您看一级市场，是看估值，还是看"能否上市、能否接住"？

③ 小米 200 倍套现离场——早期股东的退出回报

一个早期下注、数百倍退出的样本。客户里很多是企业家或家属，这个点比任何话术都能让她觉得"你懂老板的难"。

④ 高门槛 + 稀缺性——额度背后的真价值

"抢不到的才香"是人性。你能带客户看穿：稀缺不一定是价值，也可能是情绪的溢价。

3. 你跟客户聊资本，可以用的一个框架

同一个案例，你带客户从浅到深，是四层递进，不是四个层级评判。目的是让她觉得"你带我一层层看透了"：

  1. 先聊事件：追觅开 Pre-IPO 窗口，700 亿估值、3.5 亿门槛，机构抢着进。——破冰，谁都接得上。
  2. 再聊逻辑：为什么敢设高门槛？硬科技 + 出海 + 自研马达，故事够硬，所以稀缺变抢手。——展示你懂行，不止追热点。
  3. 聊资本视角：高位融资、少稀释、强议价；早期股东（小米）200 倍退出——结构比估值更关键。——展示你深，能看到老板的难。
  4. 聊资本与权力：Pre-IPO 是"入场券博弈"——谁能进、谁被挡门外，资本权力在此刻显形；高估值能不能被二级市场接住，是最终的裁判。——让客户觉得你看得透，聊到了别人聊不到的层。

4. 下次跟客户聊，你可以试着抛的几个问题

这篇没有标准话术，留几个你能自然抛出的问题：

  1. 您有没有碰过 Pre-IPO 份额？当时怎么判断值不值？
  2. 您看一级市场，是看估值，还是看"能否上市、能否接住"？
  3. 您自己的资产里，"早期股权"占比多少？流动性风险想过吗？
  4. 用刚才那个"四层框架"，您给自己的 Pre-IPO 认知打到第几层？身边很多人其实停在第一层（只看估值）。

5. 相关映射：用框架聊一个对照样本（思维训练）

跟客户聊"对照样本"，石头科技 几乎是追觅的镜像：技术派、强出海、已登陆科创板。一个已上市、一个冲刺中，刚好看"硬科技怎么变成资本"。

用第 3 节的"事件 → 逻辑 → 资本视角 → 资本与权力"框架，跟客户聊石头：

  1. 石头大众现在认知在第几层？它走到"资本与权力"这层了吗？
  2. 它的"不可替代资产"是技术还是渠道？和追觅的马达壁垒，相同不同在哪？
  3. 它的上市节奏与结构，给客户的启发是什么？早期股东赚到退出了吗？
  4. 您用这个框架，能给石头写一段"资本心智分析"吗？写得出，说明框架真用熟了。

这套框架不只对硬科技有效。任何"冲刺资本"的样本，都能这么带客户拆。练几次，您跟客户聊 Pre-IPO 的底气会不一样。

6. 我读完后自己的一点想法

写完这篇，我最触动的不是 700 亿，而是它反过来印证了那句没说出口的逻辑：一个人最值钱的，从来不是被看见，是被需要且不可替。追觅被市场需要，是因为那颗马达真的不可替代——你能带客户看到这一层，你们的关系就稳了：你不是卖产品的，你是那个能跟她对等聊资本世界、聊得穿估值与稀缺的人。这，才是你跟她之间真正值钱的连接。
