蔚来：造车新势力的融资循环与生死腾挪

案例背景｜案例来龙去脉：一家靠"融资—造车—再融资"活下来的公司，怎么在至暗时刻被一座城市救回

先把这个故事的骨血讲透。没有它，后面的"视角"都是空中楼阁——而这个故事最有价值的，恰恰是它怎么在离破产只差一步时，靠一座城市的 70 亿和一套融资循环活了过来。

1.1 来路：李斌的"造车梦"与多轮融资（2014–2017）

2014.11，李斌创立蔚来，讲的是一个"用户企业"的故事——不只卖车，是建社群、做服务。早期融资极其豪华：2015 天使轮李斌个人出资 1.5 亿美元，腾讯、高瓴、红杉等入局；此后多轮，股东名单拉到腾讯、高瓴、红杉、淡马锡、Baillie Gifford、愉悦资本等。它用 EP9 超跑、NIO Day 发布会把"高端 + 社群"的调性先立起来——先让资本和用户"信"，再谈交付。

1.2 蜕变：美股上市，中国新能源车赴美第一股（2018）

2018.9.12，蔚来在纽交所上市（NIO），发行价 6.26 美元，募资约 10 亿美元，成为"中国新能源车赴美第一股"。它刻意和特斯拉差异化：特斯拉拼技术与降价，蔚来拼"用户企业"——NIO House 体验空间、一键加电、社群运营，把车做成"生活方式会员"。这套叙事在资本市场很吃香，但也意味着：它比传统车企更烧钱。

1.3 上线即爆：ES8 / ES6 交付，却很快坠入至暗（2018–2019）

2018 ES8 交付、2019 ES6 上市，销量起来，但隐患同时爆发：交付瓶颈、自燃召回、现金流急速告急。股价从上市约 6 美元一路跌到 2019 年 10 月最低的约 1.19 美元，市场流传"离破产只差一个月"。一家上市才一年的明星车企，差点死在产能和现金上。

1.4 成长弧：合肥 70 亿纾困，绝地反转（2020）

转折点来自一座城市。2020.4.29，合肥市建设投资控股、国投招商、安徽高新产投等战略投资者，向蔚来中国投资 70 亿元人民币，换取 24.1% 股权，蔚来中国估值约 290 亿元，总部落地合肥；李斌保留控制权。这笔钱把蔚来从死亡线拉回。效果立竿见影：股价从不到 2 美元，涨到 2021.1 最高的 66.99 美元——合肥"抄底"成了中国地方政府风投的经典神话。

1.5 资本弧：美股 + 港股双重主要上市 + 中东资本接力（2021–2023）

活过来后，蔚来把融资通道开得更宽。2022.3.10，在港交所双重主要上市（9866.HK），采用"介绍上市"、不募资，成为"蔚小理"中又一家聚齐港股的公司——相当于给美股投资者开了一扇退出的备份门。2023，中东资本接力：阿布扎比投资机构 CYVN 先投 11 亿美元（6 月），再投 22 亿美元（12 月），成为第一大股东，但李斌保持最多投票权，CYVN 持股约 20%+。多轮融资循环，撑住了它持续的研发与换电投入。

1.6 竞争弧：它赢在"用户企业"叙事与换电生态

对特斯拉：蔚来走高端社群 + 换电，不正面拼降价。对"蔚小理"：理想押增程、小鹏押智驾，蔚来押换电 + BaaS（电池租用）。它把换电网络向第三方开放，长安、吉利等品牌已接入——硬件微利、靠后端服务与生态赚钱，像"打印机便宜、墨盒赚钱"的模型。这条路的争议一直很大：重资产、回报慢。

1.7 悬念：盈利拐点与现金消耗

据公开报道，蔚来在 2025 实现单季度首次盈利（约 12.5 亿元）——成立 11 年第一次——但全年仍承压，研发与换电的持续大额投入没停。截至 2026.7 股价约 4.78 美元，远低于 2021 年高点。一个问题悬在这里：靠融资循环撑起来的车企，到底能不能真正跑通盈利、不再依赖下一轮？

▼ 分支 A · 向内做企业（视角向内）

0. 你为什么读这个：把别人的"融资循环"，变成自己的现金流清单

蔚来最值钱的不是车，是它"在快没钱时，总能融到下一轮"的能力。

读它，对你做企业的人，价值不是"学造车"，是"学续命"。我自己的一个感受是：很多做企业的人卡在"只会赚钱、不会融资"——行情好时无所谓，一到至暗就断流。蔚来这个案例，把"怎么用叙事 + 资产在快死时换来救命钱"摊开了，你对照的，应该是自己的现金流安全垫，不是它的车。

2. 蔚来这一步步，对你做企业意味着什么

把蔚来当成主角，看它每一步动作背后的取舍。对你最有用的，不是"它多会烧"，而是"它怎么在没米下锅时还能借到米"。

① 早期多轮融资 + 高端叙事：先让人信的能力

它上市前就把"用户企业"的故事讲透，资本才肯在没交付时下注。很多做企业的人产品没影就谈估值，或有了产品却讲不清"为什么是你"。对你：你有没有一个"让人先信"的叙事？在没钱时，叙事本身就是融资资产。

② 至暗时刻的现金流管理：差一个月破产靠什么续命

2019 它离出清只差一步，靠的是此前铺好的股东关系和可抵押的产能。很多做企业的人不设现金缓冲，潮水一退就裸泳。对你：你的现金够撑几个月？至暗时靠什么续命？

③ 合肥纾困：把总部 / 资产落到一个城市换资源

它用"中国总部 + 产能落地合肥"换来了 70 亿。很多做企业的人只盯着财务投资人，看不见地方政府也是"带资源的股东"。对你：你愿不愿意用股权，换地方政府的钱、地、场景？

④ 双重上市 + 中东资本：融资通道要多开几条

美股、港股、中东主权基金，它把退路和钱路都铺好。很多做企业的人只靠一条融资路，一断就死。对你：你的融资通道有几条？只靠一条会不会断？

3. 你自己的企业，可以对照这四步动作（切口 / 资产 / 控制权 / 跃迁）

同一个案例，落到"怎么做企业"上，是四个可执行的动作，不是四个抽象层级：

  1. 切口：你有没有一个"让人先信"的叙事（像"用户企业"）？在没钱时，叙事就是你的融资资产，别只埋头做产品。
  2. 资产：你真正能抵押 / 入股的"硬资产"是什么——工厂、技术、数据、还是用户社群？蔚来用"中国总部 + 产能"换了合肥 70 亿。
  3. 控制权：多轮融资后你怎么保住话事权？李斌让出股份、却用投票权设计留住控制——控制权是谈出来的，不是股份自然给的。
  4. 跃迁：从"一家烧钱公司"跳到"被城市 / 国家资本背书"，融资通道从市场到政府到中东——多元才活，单一必断。

4. 合上文章，想想你自己的四件事

这篇没有标准答案，留四个问题给你自己：

  1. 你的"让人先信"的叙事是什么？投资人 / 合作方为什么先信你？
  2. 你的现金缓冲够撑几个月？至暗时刻靠什么续命？
  3. 你愿不愿意用股权换地方 / 战略资本？代价你算过吗？
  4. 你的融资通道有几条？只靠一条，会不会在某个时点突然断？

5. 相关映射：用框架看一个"没跑通循环"的样本（思维训练）

和蔚来形成残酷反照的是 威马汽车：同样是第一波造车新势力，却没能跑通融资循环，最终陷入困境、申请破产重整。用第 3 节的"切口 / 资产 / 控制权 / 跃迁"四步，去拆威马（或你自己的企业）：

  1. 它的"切口"也是新能源，为什么叙事没能持续换来钱？和你可能的差距差在哪？
  2. 它的"资产"问题在哪——重资产工厂是不是变成了包袱而非筹码？
  3. 它的"控制权 / 融资节奏"出了什么错？和蔚来"多通道续命"对照，你更想避开哪条路？
  4. 如果用这四步写一段"我的企业现金流诊断"，你会怎么写？最该补的是哪一步？

这套四步不只对车企有效。任何"靠融资循环长大"的企业，都能这么拆一遍自己的现金安全垫。训练几次，你看生死的眼光会不一样。

6. 我读完后自己的一点想法

写完这篇，我最触动的不是它冲到 67 美元，而是那句藏在背后的逻辑：一家企业最值钱的，从来不是被看见，是被需要且不可替。蔚来之所以能在至暗被合肥、被中东资本接力救回，是因为它成了"用户愿意社群化追随、城市愿意下注"的节点——多方都需要它，才有人愿意在它快死时接盘。这中间的认知转折，才是它真正的护城河：你做自己的企业，最该守住的，也是这句——先把"被需要且不可替"的那一点磨出来，再谈融资和规模。

▼ 分支 B · 向外连客户（视角向外）

0. 你为什么读这个：用一个好案例，跟客户聊到一起去

高净值客户不缺一个"卖产品的人"，她缺一个能跟她在同一个资本语境里聊世界的人。

你的门户有一句客户证言："以前看财报像天书，现在能跟家里管钱的先生平起平坐地聊资产配置了。"这句话泄露高净值女性最深的两层心理：第一，她长期被"降维"对待——先生、banker、顾问都默认她不懂；第二，她渴望的不是被服务，是被"平视"。蔚来这个案例，我的一个感受是：它是一张体面获客的入场券——从"合肥 70 亿抄底"到"中东主权基金接力"，你能带她看透"企业生死背后的资本逻辑"，证明你和她在同一认知语境，而不是靠推销硬闯。

2. 这个案例里，有哪些点能让你跟客户聊出深度

把蔚来当成素材，看哪些点能让你跟客户从破冰聊到深。不是背数据，是借它的故事，露出你的认知。

① 合肥 70 亿"抄底"——地方政府怎么当风投

很多客户只把政府当监管者。你能接住这句话，带她看：合肥用 70 亿换 24.1% 股权 + 总部落地，赌成了风投神话。这不是八卦，是让她知道你看得比热搜深——你聊的是"地方资本的逻辑"，不是"救了一家车企"。

② 股价从 1 美元到 67 美元——极端波动里的机会与风险

这个过山车特别适合跟有股价的客户聊：同一家公司，一年半涨 50 倍又跌回去。你能带她看"故事"和"现金"怎么共同决定价格，比任何理财话术都让她觉得你看得透。

③ 李斌保控制权——老板怎么在多次融资后还说了算

客户里很多是企业家或家属，这个"让股份、留投票权"的故事，比任何话术都能让她觉得"你懂老板的难"——她自己的企业，融资后话事权还在吗？

④ 换电开放——重资产怎么变成生态

蔚来把烧钱的换电网络开放给别家，把成本项变生态项。这点很适合跟"重资产怕套牢"的客户聊：你的重投入，有没有可能变成别人也来用的网络？

3. 你跟客户聊资本，可以用的四层框架（事件 → 逻辑 → 资本视角 → 资本与权力）

同一个案例，你带客户从浅到深，是四层递进，不是四个层级评判。目的是让她觉得"你带我一层层看透了"：

  1. 事件：蔚来一度跌到 1 美元，合肥 70 亿救回，又冲到 67 美元。——破冰，谁都接得上。
  2. 逻辑：它活下来靠"融资循环 + 用户叙事"，不是单一产品能打——展示你懂行，不止追热点。
  3. 资本视角：合肥用 70 亿换 24.1% 股权 + 总部，双赢；后来中东资本接力。资本买的是"不被出清"的期权。——展示你深，能看到老板的难。
  4. 资本与权力：当一家企业被城市和国家资本同时背书，它的权力来自"被多方需要"——这是它至暗还能续命的根。——让她觉得你看得透，聊到了别人聊不到的层。

4. 下次跟客户聊，你可以试着抛的几个问题

这篇没有标准话术，留几个你能自然抛出的问题：

  1. 您觉得合肥 70 亿是"抄底"还是"接盘"？如果是您，会在企业至暗时下注吗？
  2. 您自己的企业 / 家族资产，融资后控制权还在您手里吗？怎么设计的？
  3. 用刚才那个"四层框架"，您给自己的资本认知打到第几层？身边很多人其实停在第一层（只看股价涨跌）。
  4. 如果一项重资产投入能开放成生态（像换电），您的"重"有没有可能变"轻"？

5. 相关映射：用框架聊"风投之城"的打法（思维训练）

跟客户聊"对照样本"，合肥的"风投之城"模式几乎是蔚来案例的放大版：京东方（面板）、长鑫（存储）、蔚来（整车），合肥用地方政府资本一连押中几条硬科技赛道。用第 3 节的"事件 → 逻辑 → 资本视角 → 资本与权力"框架，跟客户聊合肥：

  1. 合肥的"事件"层：它怎么从"最不出名省会"变成"风投神话"？
  2. 它的"逻辑"层：政府当投资人，赌的是产业集聚还是单笔回报？
  3. 它的"资本视角"层：70 亿换来的，是股权还是一座城市的产业链？
  4. 它的"资本与权力"层走到哪了？您用这个框架，能给"地方资本"写一段分析吗？写得出，说明框架真用熟了。

这套四层框架不只对蔚来有效。任何"企业生死 + 资本接力"的新闻，你都能带客户这么拆。训练几次，你跟客户聊资本的深度会不一样。

6. 我读完后自己的一点想法

写完这篇，我最想跟客户分享的不是股价涨跌，而是那句：一家企业最被记住的，不是它多大，是它"被需要且不可替"到了哪一层。蔚来在至暗能被一座城市、一个主权基金接力救回，靠的是它成了多方都需要、都舍不得它死的节点。你带客户聊透这个，她记住的不是几个数字，是"你带我把一个生死故事看透了"。你不是卖产品的，是能和她平起平坐、聊资本世界的人——这，才是这个案例真正帮你连上的东西。
