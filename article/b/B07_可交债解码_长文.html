可交债解码：不用减持也能高位套现

Exchangeable BondCapital DecodingLong-form
你在服务客户时，大股东说"我发个可交债"——你第一反应是什么？是"又融资了"，还是"他想卖股票了"？可交换公司债券（Exchangeable Bond, EB）最妙的地方在于：它表面是一只债，骨子里是一把"带期权的高级尺子"——大股东把手里已有的股票质押出去借一笔钱，约定到期投资者要么拿回本息、要么把债券换成那批股票。涨了，投资者来换股，大股东变相"高位卖了股票"还白用了几年低息资金；跌了，投资者不换股，大股东还本付息保住股票，等于做了一笔便宜的融资。减持和融资，一券两用。本篇把可交债的套利逻辑拆开：它和可转债到底差在哪、换股比例和下修/赎回条款怎么设计、转股期里发行人和持有人怎么打架、券商承销时怎么在两边之间磨条款——最后用一个脱敏案例，把"换股溢价20%+下修触发+持有人转股vs不转股"算成一张你能直接拿去给客户讲的数字推导表。
可交债 = 存量股质押 × 低息融资 + 看涨期权
大股东用"已经持有的股票"做抵押借一笔钱；到期持有人要么拿钱、要么换走股票。对大股东：涨了变相高位减持、跌了白用低息资金——减持与融资，一个工具两头占。

一、基础认知盘

可交债不是"又发了一只债"，是股东把"未来可能卖出的股票"先抵押出去借了笔钱

可交换公司债券（可交债 / EB）：上市公司股东把自己持有的股票"打包"成债券卖给投资者。大白话——你手里有一批值钱股票，但舍不得现在卖（怕砸盘压价、怕缴税、怕触发减持披露和比例限制），于是把这批发债抵押出去借一笔钱，约定"到期你可以选择拿钱，或者拿这批股票抵债"。专业定义：依据《上市公司股东发行可交换公司债券试行规定》（证监会公告〔2008〕41号），上市公司股东依法发行、在一定期限内依据约定条件可交换成该股东所持有的上市公司股票的公司债券。

为什么大股东爱用这一招：直接在二级市场减持 = 立刻砸盘压价、还要走减持披露和每90天不超过1%等比例限制、接盘方也难找；发可交债 = 先把钱拿到手，等股价高位时再"换股"把股票过户给持有人（变相减持），过程温柔、还能白用几年低息资金。一句话：它把"减持"这件容易引发市场恐慌的事，包装成了一场有秩序的、带期权属性的融资。

可交债 vs 可转债，一张表先建立直觉

  • 发行人是谁：可交债的发行人是股东（大股东）；可转债的发行人是上市公司本身。
  • 换的是什么股：可交债换的是发行人持有的存量股（股票本来就存在）；可转债换的是公司新发的增量股。
  • 总股本变不变：可交债换股不增加总股本（只是股东换人，股本盘子不变）；可转债换股增加总股本、摊薄每股收益。
  • 钱进谁的口袋：可交债募资进了大股东口袋（是他个人/持股平台在融资）；可转债募资进了上市公司账户。
  • 一句话定位：可交债是股东的减持 + 融资工具；可转债是上市公司的融资 + 转股工具。

停下来，先想一个问题

你下次听到大股东说"我发个可交债"，第一反应不该是"又融资了"，而是"他到底想融资，还是想减持？"——这两个目的设计的条款天差地别。模块一最该带走的认知：可交债是一把双用途尺子，它既能量出大股东"缺钱"的程度，也能量出他"想不想走"的诚意。

可交债与可转债，五个维度核心差异一表速查

二、规则与实务深水区

换股比例怎么算、下修怎么触发、赎回怎么逼换股、券商怎么设计条款——逐一拆开

换股比例与换股溢价率

公式：换股比例 = 债券面值100元 ÷ 换股价。例：换股价12元 → 每张债券换 100÷12 ≈ 8.33股。

换股溢价率 =（换股价 − 发行时市价）÷ 发行时市价。换股价12元、市价10元 → 溢价率 =（12−10）÷10 = 20%。意思是：投资者要等股价涨过12元（比现在高20%）换股才划算。

溢价率高低 = 大股东的"真实意图温度计"：溢价 20%以上（高溢价）→ 投资者很难换股，大股东偏"融资型"，压根不想真让股票被换走；溢价 平价或折价 → 一上市就接近换股临界，大股东偏"减持型"，借机套现。

下修条款（向下修正换股价）

规则：当正股价格连续N个交易日低于换股价的一定比例（常见85%、80%），发行人可（注意是"可"，非"必须"）向下修正换股价，使换股更有吸引力。该条款属募集说明书里的市场化约定，无强制触发义务。

实务 + 怎么操作：下修对持有人有利（更容易换股赚钱），但下修后每张债能换更多股、需要更多股票覆盖换股，发行人要补充质押。大股东下修的动机 = 想促成换股（把股票换出去=减持）；选择不下修 = 想保住股票（到期还本）。判断诀窍：看下修触发后大股东"动不动手"——动手就是想走，不动就是想留。

赎回条款（逼持有人换股的最强武器）

规则：当正股价格连续N个交易日高于换股价的一定比例（常见130%），发行人有权按面值+应计利息赎回债券。另有到期赎回（还本付息）。

实务 + 怎么操作：赎回等于"逼宫"——投资者如果不换股、等着被赎回，拿回的只是面值+微息（远少于此时股票的价值），理性选择是赶紧换股。所以大股东在股价高时行使赎回，把"还钱义务"变成"过户股票"，变相高位减持。赎回条款是发行人掌握主动权的核心抓手。

回售条款（持有人的安全带）

规则：若正股持续大跌、下修也救不回来，持有人可在回售期按面值+利息把债券卖回给发行人。这是保护投资者本金的"最后一道闸"，也是约束大股东不能随意摆烂的压力。

实务：回售把"股价跌、大股东不还本"的风险封死——一旦触发回售，大股东必须拿出真金白银接回债券。所以担保质押比例和回售线，是投资者敢买可交债的底气。

承销视角 · 券商怎么设计条款

券商是承销商，要两头平衡：大股东想低利率、高溢价（少稀释、易融资），投资者想要安全性（质押足、有下修/回售保护、换股有赚头）。依据《证券发行与承销管理办法》《公司债券发行与交易管理办法》（证监会令）开展承销。

设计五维 + 怎么操作：①换股溢价率（意图的温度计）；②票面利率（融资型常低至0.5%~1%，减持型可能略高以吸引认购）；③期限（1~3年，减持型偏短、融资型偏长）；④下修 / 赎回 / 回售触发线（决定博弈主动权）；⑤担保质押比例（通常质押130%~150%的股票，由中国结算办理质押登记）。

操作动作：先问大股东"这债是融资还是减持"——定调后倒推条款；再拿条款去路演问投资者接受度，两边磨合。融资型条款大股东舒服、投资者要担保足；减持型条款投资者舒服、大股东让利。

停下来，先想一个问题

现在你手里有一只可交债的募集说明书：换股价溢价20%、票面利率1%、期限3年、质押150%、下修"可"但不强制。这更像是融资还是减持？模块二最该带走的认知：条款不是随便写的，每一条都是大股东"想不想走、想怎么走"的密码——溢价率、利率、期限、下修意愿，四个旋钮转一转，意图就显形了。

三、信号解码与判断框架

看到大股东发可交债，不急着欢呼或恐慌，三步判断：信号→博弈→判断

第一步 · 看到什么信号

  • 标的：换成哪只股票、大股东持股多少、质押多少、质押比例够不够覆盖换股。
  • 期限：1年 / 2年 / 3年——越短，越像"急着把股票换出去"。
  • 换股价 + 溢价率：溢价20%以上偏融资；平价 / 折价偏减持。
  • 票面利率：超低利率（0.5%~1%）偏融资型（大股东想低成本拿钱）；给到吸引力利率偏想让投资者安心持有到期。
  • 下修 / 赎回设定：易触发下修 + 强赎回 = 大股东想促换股；难下修 = 想保股票。

第二步 · 背后什么博弈

每个信号背后都是四方在"换不换股"这一点上拉扯：

  • 发行人（大股东）：想高位套现或低息融资，立场"往上推股价、促换股"，掌握下修/赎回主动权。
  • 持有人（投资者）：想保本+赚换股价差，立场"股价够高才换、不够就拿利息"，靠回售保底。
  • 监管 / 市场：关注是否变相绕道减持、质押会不会爆仓，立场"防绕道、防违约"。
  • 正股散户：怕大股东借可交债"高位出货"，情绪偏空；但可交债不直接砸盘，冲击小于集中竞价减持，短期反而是"缓释"。

四方博弈最终在"换股"这一个动作上交汇：大股东想换（减持/消债）、持有人想挑时机换（赚钱）、监管盯着别出风险。转股期就是这场博弈的主战场——进入换股期后，大股东有动力"做市值"推高股价促成换股，也可能压低股价避免被换走，利益冲突极其明显。

第三步 · 判断方向

怎么识别是"高位套现"还是"低位让利 / 低息融资"：

  • 高位套现型信号：低溢价 / 折价 + 短期限 + 易下修 + 高质押比例 + 强赎回 → 判断：大股东想借换股变相减持。正股短期有"促换股做市值"的动力，但中长期减持预期偏空，散户情绪承压。
  • 低息融资型信号：高溢价（20%+）+ 低利率 + 难下修 + 保留股票 → 判断：大股东不想卖，就是缺钱用低成本融资。对正股中性偏稳，质押是安全垫不是抛压。
  • 判断动作：作为券商客户经理或私行顾问，你不需要会算这只债的定价，但要会看"溢价率 + 期限 + 下修意愿"三个旋钮——它们直接告诉你客户这把是融资还是减持，这才是你能在饭局上镇住场、建立专业信任的关键。

停下来，先想一个问题

下次客户递给你一只可交债方案，你第一句话不该是"利息多少"，而是"大哥，这债您是想借点钱花，还是想把股票换个高价走？"——把溢价率、期限、下修意愿三个旋钮一转，意图就藏不住。模块三最该带走的是：可交债是信号，信号背后是博弈，博弈决定它是套现还是融资——你不是算债的人，你是读意图的人。

四、脱敏案例：某大股东发可交债推演

把换股溢价20% + 下修触发 + 持有人转股vs不转股，算成一张数字推导表

公司画像：某制造业上市公司，总股本 3亿股，大股东持股 1.2亿股（占40%），当前股价 10元。大股东想"既拿到一笔低成本资金，又给自己留一条高位减持的后路"，于是发行可交债。

可交债条款设定：规模 9亿元，面值100元、共 900万张，期限 3年，票面利率 1%/年；换股价 12元（相对现价溢价20%）；换股比例 = 100÷12 ≈ 8.33股/张；全部换股需 7500万股；担保质押比例 130% → 初始质押 9750万股。下修触发线 = 换股价×85% = 10.2元；赎回触发线 = 换股价×130% = 15.6元。

情景正股走势 / 触发投资者决策大股东所得投资者所得对正股信号
上行 · 触发赎回股价涨至 16元 > 15.6元赎回线被赎回"逼"换股变相以12元/股减持7500万，得9亿现金+白用3年（利息仅2700万）换7500万股市值12亿，浮盈 3亿股本不变，大股东40%→15%，偏空情绪（高位减持落地）
横盘 · 不触条款股价维持 10元换股即亏，持有到期还本9.27亿，保留1.2亿股，省利息≈1.2亿（对比5%信用融资成本）拿回本息9.27亿，收益率低无直接冲击，中性偏稳
下行 · 不下修股价跌至 8元 < 10.2元下修线，大股东不修持有到期还本9.27亿，保住股票，低息融资3年拿回本息9.27亿质押预警但无抛压，大股东资金成本极低
下行 · 下修促换下修换股价至 9元，股价仍8元仍不换（需>9元才划算）若股价回9元+可促换股变相减持；当前需补质押至1.3亿股（大股东1.2亿不足）保本都难盈利担保不足风险暴露，下修也未必促成换股

这张表说明什么

  • 大股东稳赚不赔的底层逻辑：涨了（情景一）以12元/股变相减持，比发债时市价10元多套现（12−10）×7500万 = 1.5亿，还白用3年1%低息资金；跌了（情景二、三）还本付息保住股票，做的是一笔比信用贷便宜得多的融资。两头占。
  • 持有人的胜负手在"换股价"：只有股价涨过换股价、且不被赎回"逼"在低位换，才赚得到价差；否则就是一笔低息存款。
  • 下修不是万能：情景四显示，股价跌太深，即便下修，若仍低于新换股价，持有人仍不换；而且下修要补质押，大股东股票不够还会暴露担保不足风险。
  • 对正股的真实影响：可交债本身不增加总股本（换股是存量股过户），比集中竞价减持温柔；但"大股东想高位走"的预期会压制散户情绪，属于缓释型、预期型压力，不是即时砸盘。

核心结论：可交债的套利不是"稳赚"，是"把减持的择时权和低成本融资权打包"。大股东在溢价20%、低利率、3年期的设定下，给自己留了两条路：股价上去就高位套现，股价下来就低息借钱。作为服务股东的你，读懂这张推导表，就能在方案设计阶段帮客户把"想融资还是想减持"想明白、把条款拧对方向。

停下来，先想一个问题

如果你就是那位大股东，发债时股价10元、换股价定12元——你希望3年后股价涨到16元被赎回换股，还是希望横盘10元还本保股？答案暴露了你的真实意图：想套现就盼涨（触发赎回），想融资就盼平（白用钱）。模块四最该带走的：大股东自己都不一定说清的意图，藏在换股价和期限里。

五、纵深三棱镜

联动疆界 · 风险盲区 · 资本运作视角——三面镜子互相关联，点开逐个看

联动疆界（概要）：可交债上接定增B06（同为股东融资/减持工具——定增是"低位拿货"、可交债是"高位套现"，方向相反却常常配合使用），下接大宗B08（可交债换股后，持有人拿到的股票往往通过大宗交易或二级市场减持，是可交债的"下游出口"）。三个工具串成一条"股东股份进出"的资本链。

风险盲区（概要）：①下修稀释/补质押风险——下修要更多股票覆盖换股，大股东股票不足就暴露担保缺口；②履约担保风险——股价跌至质押预警线，大股东有补质押/补保证金义务，资金链紧时会爆；③标的下跌风险——股价长期低迷，持有人不换股，大股东到期集中还本，形成现金流压力。

资本运作视角（概要）：可交债是减持的优雅替代——不砸盘、锁定价、低息融资三合一。集中竞价减持受比例和披露约束、容易引发恐慌；可交债把"卖股票"变成"发债+换股"两步曲，过程有序、价格提前锁定、还能先拿钱。它是大股东在合规框架内管理股份流动性的高阶工具。

六、三层进阶

三步不必全做，取你当前最需要的那一层

第一阶 · 看懂门道　遇到可交债，先记住一句话：它是大股东用存量股质押借的钱，换股=变相减持。区分可交债（股东发、换存量股、不稀释）与可转债（公司发、换增量股、稀释）是基本功。

第二阶 · 能用判断　看到一只可交债，拧一拧"溢价率 + 期限 + 下修意愿"三个旋钮：高溢价+长期限+难下修=融资；低溢价+短期限+易下修=减持。能对客户说清"这债大哥是想借钱还是想走"。

第三阶 · 精通实务　①帮股东定调"融资型vs减持型"再倒推条款②算清换股比例、质押担保比例（130%~150%）和补质押缺口③预判下修/赎回触发后的博弈走向④把可交债与定增B06、大宗B08串成股东股份进出的一条链⑤在方案设计阶段就帮客户把"想融资还是想减持"想明白、把条款拧对方向。

可交债 vs 可转债 核心差异

维度可交换债券（EB）可转换债券（CB）
发行人上市公司股东（大股东/持股平台）上市公司本身
换股标的发行人持有的存量股（已存在）公司新发增量股
总股本换股不增加总股本（股东换人）换股增加总股本、摊薄EPS
募资去向进大股东口袋进上市公司账户
核心目的股东减持 + 融资公司融资 + 转股
质押担保必须以标的股票质押（中国结算登记）一般无股票质押，靠信用/担保
对正股影响不稀释，但释放"股东想走"预期稀释每股收益，摊薄存量股东

一句话记忆：可交债是"股东拿旧股换钱"，可转债是"公司发新股换钱"。两者都是"债+转股权"，但一个动存量、一个动增量，决定了它们对股本、对股东、对市场的冲击完全不同。

联动疆界

三棱镜第一条棱：可交债不是孤立工具，它上接定增、下接大宗，串成一条"股东股份进出"的资本链。三个工具方向相反却常常配合使用。

上接定增 B06 · 怎么联动、怎么操作

定增是"低位拿货"——大股东或关联方在股价低位参与定向增发，拿到便宜筹码；可交债是"高位套现"——把高位股票通过换股变相减持。操作上：

  • 低位时定增/增持拿筹码，高位时发可交债锁定出货价格，一进一出完成"低买高卖"的股份周转；
  • 定增有锁定期，可交债有换股期，两者期限可错配，让股东在合规框架内管理流动性；
  • 关键差异：定增是"公司融资、股本增加"，可交债是"股东融资、股本不变"——用途和冲击完全不同。

下接大宗 B08 · 怎么联动、怎么操作

可交债换股后，持有人（投资者）拿到的是存量股——这些股票往往通过大宗交易或二级市场减持，是可交债的"下游出口"。操作上：

  • 持有人换股获得股票后，若在二级市场直接抛会砸价，更倾向于走大宗交易（折价撮合、找接盘方）；
  • 所以可交债的"减持效应"常常滞后体现在大宗交易的放量上——你看大宗异动，要往上游想"是不是某只可交债换股了"；
  • 折价率规律：大宗接盘方要安全垫，可交债换股后的抛压会压低大宗折价率。

上接定增（低位拿货）、本篇可交债（高位套现）、下接大宗（出货出口）——三条线在"股东股份流动性管理"这个节点交汇：进、持、出，各有工具，逻辑一脉相承。

风险盲区

三棱镜第二条棱：联动疆界讲"连得出去"，风险盲区讲"看不见的坑"。可交债看着温柔，坑往往出在质押和还本上。

盲区一 · 下修稀释与补质押缺口

  • 下修换股价 → 每张债能换更多股 → 需要更多股票覆盖换股。原质押130%可能不够，大股东必须补充质押；
  • 若大股东持股本就接近质押上限（案例中1.2亿股、下修后需1.3亿股），下修会直接暴露担保不足，引发违约担忧；
  • 修正方法：设计时就预留"补质押空间"，或把下修幅度限制在担保覆盖范围内。

盲区二 · 履约担保与质押爆仓

  • 股价跌至质押预警线，大股东有补质押 / 补保证金义务；
  • 若大股东资金链紧张、无股可补、无钱可补，质押股票可能被强制处置，形成"雪崩式"抛压；
  • 投资人视角：买可交债不能只看换股收益，要看质押比例够不够厚、大股东偿债能力——担保才是本金的安全垫。

盲区三 · 标的下跌与集中还本

  • 股价长期低迷，持有人不换股，可交债到期变成"纯债"——大股东要集中还本（案例9亿+利息）；
  • 若同时有多只可交债 / 其他债务到期，现金流压力叠加，低息融资的"便宜"瞬间变"贵"；
  • 回售条款更是放大器：股价跌透、下修救不回，持有人行使回售，大股东必须真金白银接回债券。

三个盲区共同指向一句话：可交债的"低息"是用"质押和还本义务"换来的，担保不足和集中还本，才是真正要盯的风险。

资本运作视角

三棱镜第三条棱：把可交债放回"股东怎么管理股份流动性"的大图景——它最优雅的地方，是减持的温柔替代。

为什么可交债是"优雅的减持"

  • 不砸盘：直接二级市场减持 = 立刻压价、引发恐慌；可交债先发债拿钱，换股是存量股过户，不直接制造卖盘；
  • 锁定价：换股价提前锁定，等于大股东给自己定了一个"最低出货价"（溢价越高、底价越高），比随行就市卖更有掌控感；
  • 低息融资：发债拿到现金，白用几年1%左右低息资金，比信用贷、质押回购都便宜；
  • 合规绕道：集中竞价减持受每90天不超1%、提前披露等约束，可交债在合规框架内把"卖股票"变成"发债+换股"两步曲，过程有序。

怎么操作 · 服务股东的实务动作

  • 先问清客户"融资还是减持"定调，再倒推换股溢价率、期限、利率；
  • 减持型：低溢价 + 短期限 + 易下修 + 强赎回，帮客户在高位有序出货；
  • 融资型：高溢价 + 长期限 + 难下修，帮客户低成本拿钱、保住股份；
  • 协同定增B06（低位拿货）与大宗B08（换股后出货），把可交债放进股东完整的股份周转链条里设计。

可交债的本质，是给大股东一个"进可高位套现、退可低息融资"的期权。读懂它，你就读懂了股东在合规框架内管理流动性的高阶玩法。
