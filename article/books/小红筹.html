小红筹：红筹架构实务

红筹架构实务指南 · 写给要把境内权益搬去境外资本市场定价的创业者与创始人

关于“怎么绕开境内监管、把私有权益装进境外架构去上市”的第一性原理。本卡是「资本人物馆·IPO 与资本运作书架」的精读认知卡：先讲清这本不可替代的增量，再拆成 13 个认知单元细读，最后用读书 map 把线织成网、形成闭环。

书架② · IPO 与资本运作书卡 · 精读认知闭环红筹 / VIE / 10号文 / 37号文跨境资本架构
本卡为精读框架，非原书替代。观点与金句来自原书；外链媒体请在合规平台观看，注意信息甄别。请读者独立判断，决策自负。
为什么单读这本？《读懂 IPO》讲境内 A 股路径、《Venture Deals》讲拿美元前的条款——而《小红筹》独有序增量是两点：①它把“境外上市”从“找投行”还原成“搭结构”（10 号文怎么绕、VIE 怎么控、37 号文怎么登记）；②它讲清“小红筹”(民企)与“大红筹”(国企)的本质区别，以及结构的每一步都是“监管套利 vs 合规”的走钢丝。这本书不可替代，是因为它专门解决“我的业务在境内，钱和定价权却要去境外”这个民企最现实的盲点。
读前 · 为什么是这本？
Activate — 先戳中你的问题

你不是缺“去哪上市”的选择，你缺的是“我的架构能不能在境内监管下活下来、在境外被定价”的判断。

读前自检
你的业务是否在外资禁入 / 限制行业(需 VIE)？你或联创的境外身份 / 返程投资登记做了没？两个都模糊——这本书是给你写的。
精读拆解 · 13 个认知单元
Deep Read — 每个单元：论证链 → 反直觉 → 金句 → 你的情境
单元 01
什么是小红筹：民企的境外上市通道
论证链
大红筹 = 国企(招商局 / 中信)经政府批；小红筹 = 民企把境内权益通过境外 SPV(开曼 / BVI)控制、境外上市。
反直觉
你以为“红筹”是国企专利，其实民企的“小红筹”才是美元基金最爱的退出通道。
金句
小红筹，是民营企业走向境外资本市场最现实的桥。
你的情境
你是民企想拿美元——VC 要求开曼主体，小红筹是你的标准动作。
单元 02
红筹四步：设立 SPV → 重组 → 控制 → 上市
论证链
创始人设 BVI / 开曼控股公司→收购境内运营实体(返程)→协议 / 股权控制→境外发行。
反直觉
重组不是“搬家”，是“把控制权挪到境外、境内留运营”，每一步都踩监管红线。
金句
红筹的本质，是把“控制权”放到监管够不着的时区。
你的情境
你直接拿境内公司去谈美元 VC——对方要的是开曼主体，结构先做对。
单元 03
10 号文：返程并购的闸门
论证链
2006 年《关于外国投资者并购境内企业的规定》(10 号文)要求返程并购报商务部审批→阻断了“先设境外再并购境内”的老路。
反直觉
10 号文不是禁红筹，是让“简单红筹”变难；无数架构为绕它而生(信托 / 代持 / 境外身份)。
金句
10 号文之后，红筹从“流程”变成了“艺术”。
你的情境
你用境内自然人直接设 SPV 并购自己公司——卡在商务部审批，得换路径。
单元 04
VIE(协议控制)：外资禁入行业的例外
论证链
业务在外资禁入(如增值电信 / 教育)→不能股权控制→用一系列协议(独家服务 / 投票权委托 / 股权质押)实现“控制不持股”。
反直觉
VIE 是“法律上不拥有、经济上全控制”的脆弱结构，全靠合同与监管默契。
金句
VIE，是用一揽子合同在监管缝隙里搭出的控制权。
你的情境
你做在线教育(外资限入)——只能 VIE，股权直控走不通。
单元 05
37 号文：自然人境外投资登记
论证链
境内自然人境外投资 / 返程投资须办 37 号文外汇登记→否则资金出境、利润汇回都违规。
反直觉
你以为“钱出去再说”，没登记 = 外汇违规，未来上市 / 退出都卡。
金句
没有 37 号文登记，红筹的每一笔资金流动都是隐患。
你的情境
你创始人没做 37 号文登记就设了 BVI——补登记时已被外管关注。
单元 06
大红筹 vs 小红筹：审批路径不同
论证链
大红筹(国企)走国资委 / 发改委 / 证监会“红筹批复”；小红筹(民企)靠结构与 VIE 绕开，少一道批。
反直觉
国企反而“名正言顺有批文”，民企“灵活却游走在批复之外”。
金句
大红筹靠批文，小红筹靠结构。
你的情境
你是民企却想照搬国企红筹路径——没那个批复通道，得靠结构。
单元 07
新浪模式：VIE 的鼻祖
论证链
2000 年新浪为赴美上市首创 VIE(协议控制)→此后互联网中概几乎全用，成“中概标配”。
反直觉
你以为 VIE 是新发明，其实它是 20 多年前新浪被逼出来的“中国方案”。
金句
所有中概股的 VIE，都从新浪那纸协议开始。
你的情境
你做中概准备上市——VIE 不是选项，是必经结构。
单元 08
外汇与税务：结构的隐藏成本
论证链
跨境架构涉汇(37 号文 / ODI)、税(间接转让 / CFC / 预提所得税)→结构的每一步都有税与汇的成本。
反直觉
你以为“搭完架构就完了”，税务居民身份 / 间接转让一旦触发，补税可能吞掉利润。
金句
红筹省下的，可能都在税与汇里还回去。
你的情境
你境外卖了公司——间接转让被认定应税，补税 + 滞纳金超预期。
单元 09
控制链设计：BVI / 开曼 / 香港的分工
论证链
BVI(隐私 / 灵活)持顶层→开曼(上市主体 / 税务中立)→香港( treaty 优惠 / 利润中转)→境内。
反直觉
不是“随便设几个壳”，每层都有税务 / 法律 / 上市功能的分工。
金句
控制链每一层，都是为“税、隐私、上市”三件事权衡的结果。
你的情境
你少了香港层——境内分红到开曼被双重征税，白白多交。
单元 10
重组对价与税务：怎么“买”自己的公司
论证链
境外 SPV 并购境内实体需支付对价→现金 / 股权置换，触发所得税；用股权置换可递延。
反直觉
你“左手倒右手”也被征税——重组不是免费搬家。
金句
红筹重组里的每一次“倒手”，税务局都看在眼里。
你的情境
你用现金对价重组——创始人先交一大笔个税，现金流吃紧。
单元 11
合规边界：套利不是违规
论证链
红筹是“在法律框架内优化路径”，不是规避法律→一旦被认定虚假 VIE / 代持，上市即撤。
反直觉
你以为“擦边就能上”，监管对“假外资 / 假协议”零容忍，踩线即全盘崩。
金句
红筹是走钢丝，不是跳崖——边界之内才是艺术。
你的情境
你用代持掩盖返程——披露时被抓，上市进程归零。
单元 12
上市地选择：港股 / 美股 / 不上市
论证链
港股(同股同权友好 / proximity)、美股(估值高 / 中概承压)、或先不上市拿美元并购→地点决定结构与估值。
反直觉
你以为“美国最好”，中概审计 / 政治风险下，港股可能更稳。
金句
上市地，是红筹结构的终极约束。
你的情境
你冲美股却遇中概退市风险——结构得预留港股 backup。
单元 13
红筹的脆弱：政策一变，全链重搭
论证链
境内监管(10 号文 / VIE 政策 / 数据安全)一变→已搭结构可能失效，重搭成本高。
反直觉
你以为“搭好一劳永逸”，红筹是“与监管共舞”的活结构，随时要调。
金句
红筹不是一次性工程，是跟着监管呼吸的结构。
你的情境
你 VIE 所涉行业出新规——控制协议要重谈，上市时间表推迟。
冲突现场 · 当裁判
Clash — 不同作者的观点打架，你来判断

案子：一位创始人，业务在教育(外资限入)。架构师 A 劝“用 VIE，协议控制，赶紧去美股”；合规官 B 劝“VIE 政策不确定，先做 37 号文登记 + 港股预备，VIE 只作备选，别 all-in”。两人都对，风险偏好不同。

当裁判：看“监管确定性”与“退出紧迫度”。若美股窗口紧且 VIE 可控(单元 04/07)，可先 VIE；若政策风险高、想稳，港股 + 小红筹股权控制(非 VIE)更稳(单元 12)。判：列“VIE 失效触发清单”——数据安全 / 行业新规 / 审计——任一高概率，则降 VIE 权重。合上书，写你的判：你的结构是“股权直控”还是“协议控制”？再去读书 map 看“VIE / 10 号文”节点怎么连。

微观层 · 本书概念网：小红筹由六条路径汇成——红筹架构(境外 SPV)与 VIE(协议控制)是骨架(虚线并列 = 都在“搭结构”)；10 号文(返程并购闸门)与 37 号文(外汇登记)是合规双闸(虚线并列 = 都在“过监管”)；美元基金(外资 VC 偏好)与上市地(港股 / 美股)是终点约束(虚线并列 = 都在“被境外定价”)。
你的落点：高智女性创业者常卡在“10 号文 / 37 号文”两道闸——结构再美，登记与审批不过就是空中楼阁。
